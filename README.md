# 📊 Data Science do Zero
### *Noções Fundamentais com Python* — Joel Grus


---

## 📖 Sobre o Livro

**Autor:** Joel Grus  
**Editora:** Alta Books (edição brasileira) / O'Reilly Media (original)  
**Título original:** *Data Science from Scratch: First Principles with Python*  
**Linguagem:** Python 3  
**Nível:** Iniciante a Intermediário

Para dominar Data Science de verdade, não basta conhecer bibliotecas e frameworks — é preciso entender os **princípios e ideias** que sustentam a área. Este livro oferece uma abordagem única: implementar ferramentas e algoritmos **do zero**, construindo intuição sólida sobre cada conceito.

O fio condutor da obra é a empresa fictícia **DataSciencester**, uma rede social para cientistas de dados, onde o leitor é "contratado" para resolver problemas reais usando ciência de dados.

---

## 👤 Sobre o Autor

**Joel Grus** é engenheiro de software e cientista de dados. Trabalhou em diversas startups de tecnologia e no Google. Formado com PhD pelo Caltech, é um defensor fervoroso do aprendizado profundo de conceitos fundamentais — daí a proposta central do livro: entender fazendo, sem depender de "caixas-pretas".
